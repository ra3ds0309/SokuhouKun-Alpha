# 速報くんα (Sokuho-Kun Alpha)

放送局の地震速報やニュース速報のグラフィックを再現し、ライブカメラ映像と組み合わせてリアルタイムに表示するWebシステムです。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-red.svg)

## 📌 概要

「速報くんα」は、テレビ放送の「テロップ（L字型画面や速報スーパー）」の機能美に魅了された開発者が、実用的な防災ツールとして、また放送デザインの再現プロジェクトとして開発したWebベースのアプリケーションです。

YouTubeのライブカメラ映像を背景に、時計、地名、そしてカスタマイズ可能な速報テロップをオーバーレイ表示します。

## ✨ 主な機能

- **放送レベルの再現度**: UD新ゴやロダン等、実際の放送で使用されるフォントに対応した高品質なテロップ表示。
- **リアルタイム速報送信**: 設定画面から即座にメイン画面へ速報を飛ばすことが可能（BroadcastChannel API使用）。
- **マルチライブカメラ巡回**: 最大10箇所のライブカメラを登録でき、3分ごとの自動巡回機能を搭載。
- **クイックアクセスキー**: キーボードの `1`〜`9`, `0` キーで瞬時にカメラを切り替え。地震発生時の迅速な状況確認をサポートします。
- **カスタマイズ性**: 文字色、縁取りの太さ（丸縁取り対応）、フォント、巡回ON/OFFを自由に設定可能。
- **サウンドエフェクト**: 速報受信時に「あの」緊張感のあるアラート音を再生。

## 🚀 使い方

### 1. 準備
1. このリポジトリをクローンまたはダウンロードします。
2. `assets/fonts/` ディレクトリに、使用したいフォントファイル（OTF/TTF）を配置します。
3. `assets/audio/` に効果音（`interrupt-ad.mp3`）を配置します。

### 2. 実行
1. ブラウザで `index.html` を開きます。
2. 左下の ⚙（ギアアイコン）をクリックして設定画面を開きます。
3. YouTubeのライブカメラURLと場所を入力し、設定を保存します。
4. **重要**: ブラウザの自動再生制限を解除するため、メイン画面（index.html）を一度クリックしてください。

### 3. 操作
- **数字キー (1-9, 0)**: 登録したカメラへのダイレクト切り替え。
- **Sキー**: 自動巡回のON/OFF切り替え。
